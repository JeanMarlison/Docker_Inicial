Com Objetivo de Conectar um Contaier a outro.
Rede no Docker: 
É uma forma de gerencia a conexão do Docker com outros plataforma ou até por outros containers
As Networks / Redes são criadas separadas do Containes, como os volumes
-----
Tipos de Conexão:
1° Externa: Conexão com uma API de um servidor remoto
2° Com Host: A maquina que execulta o docker
3° Entre Containers
----
Tipo de redes do Docker
Bridge: Mais comum e defaul do Doker, ultilizado quand containers precisam se conectar.
Host: Permit a conexão entre um container a máquina que está o hosteando o Doncker

Listando Redes: 
docker network ls
Algumas redes já podem está criadas.
----
Para criar uma rede 
docker create network nomeRede
----
Removendo redes
docker  network rm nomeRede
----
Removendo Rede em Massa 
docker network prune
